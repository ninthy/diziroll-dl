# diziroll-dl
Diziroll sitesi için dizi indirici, eğitim amaçlı yapılmıştır.

# Kullanımı

```
$ git clone https://github.com/ninthy/diziroll-dl.git
$ cd diziroll-dl
$ python3 index.py
```

# Önemli bilgiler
Videolar ve altyazılar ```<dizi-ismi>/```'ye kaydedilir.
